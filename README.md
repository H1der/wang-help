# 小程序小王帮帮的源代码
小王帮帮 基于Taro3+React 编写的工具类小程序，目前已经上架微信，支付宝和QQ平台
## 目前已完成的功能
- 防疫出行政策查询
- 垃圾分类查询
- 油价信息
- 快递物流信息
- 亲戚关系计算器
- LPL联赛赛程信息
## 待开发功能
- 彩票开奖查询
- ......
